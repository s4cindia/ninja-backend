 PART 1: Criteria Lists for All 4 Editions

  Section 508 Edition (39 Criteria)

  Based on Revised Section 508 Standards (2017) which aligns with WCAG 2.0 Level A & AA

  Level A (25 criteria):
  1.1.1 - Non-text Content
  1.2.1 - Audio-only and Video-only (Prerecorded)
  1.2.2 - Captions (Prerecorded)
  1.2.3 - Audio Description or Media Alternative (Prerecorded)
  1.3.1 - Info and Relationships
  1.3.2 - Meaningful Sequence
  1.3.3 - Sensory Characteristics
  1.4.1 - Use of Color
  1.4.2 - Audio Control
  2.1.1 - Keyboard
  2.1.2 - No Keyboard Trap
  2.2.1 - Timing Adjustable
  2.2.2 - Pause, Stop, Hide
  2.3.1 - Three Flashes or Below Threshold
  2.4.1 - Bypass Blocks
  2.4.2 - Page Titled
  2.4.3 - Focus Order
  2.4.4 - Link Purpose (In Context)
  3.1.1 - Language of Page
  3.2.1 - On Focus
  3.2.2 - On Input
  3.3.1 - Error Identification
  3.3.2 - Labels or Instructions
  4.1.1 - Parsing
  4.1.2 - Name, Role, Value

  Level AA (14 criteria):
  1.2.4 - Captions (Live)
  1.2.5 - Audio Description (Prerecorded)
  1.4.3 - Contrast (Minimum)
  1.4.4 - Resize Text
  1.4.5 - Images of Text
  2.4.5 - Multiple Ways
  2.4.6 - Headings and Labels
  2.4.7 - Focus Visible
  3.1.2 - Language of Parts
  3.2.3 - Consistent Navigation
  3.2.4 - Consistent Identification
  3.3.3 - Error Suggestion
  3.3.4 - Error Prevention (Legal, Financial, Data)
  4.1.3 - Status Messages (WCAG 2.1 addition)

  WCAG Edition (50 Criteria)

  WCAG 2.1 Level A, AA, and select AAA

  Level A (30 criteria):
  [All 25 from Section 508 Level A, plus:]
  2.1.4 - Character Key Shortcuts (WCAG 2.1)
  2.5.1 - Pointer Gestures (WCAG 2.1)
  2.5.2 - Pointer Cancellation (WCAG 2.1)
  2.5.3 - Label in Name (WCAG 2.1)
  2.5.4 - Motion Actuation (WCAG 2.1)

  Level AA (13 criteria):
  [All 14 from Section 508 Level AA]
  1.3.4 - Orientation (WCAG 2.1)
  1.3.5 - Identify Input Purpose (WCAG 2.1)
  1.4.10 - Reflow (WCAG 2.1)
  1.4.11 - Non-text Contrast (WCAG 2.1)
  1.4.12 - Text Spacing (WCAG 2.1)
  1.4.13 - Content on Hover or Focus (WCAG 2.1)
  4.1.3 - Status Messages (WCAG 2.1)

  Level AAA (7 criteria - select important ones):
  1.2.6 - Sign Language (Prerecorded)
  1.2.7 - Extended Audio Description (Prerecorded)
  1.4.6 - Contrast (Enhanced)
  1.4.8 - Visual Presentation
  2.2.3 - No Timing
  2.4.8 - Location
  2.5.5 - Target Size

  EU Edition (52 Criteria)

  Based on EN 301 549 V3.2.1 (2021) - European accessibility standard

  This includes all WCAG 2.1 Level A & AA (50 criteria) PLUS:
  9.2.1 - Software Keyboard Access
  9.2.2 - Non-web Software Keyboard Trap

  Note: EN 301 549 uses different numbering:
  - 9.x.x.x for software
  - 10.x.x.x for non-web documents
  - 11.x.x.x for software UI

  For web content, EN 301 549 section 9 maps directly to WCAG 2.1

  International Edition (78 Criteria)

  Comprehensive - covers Section 508 + EU + additional WCAG AAA

  Includes:
  - All 39 Section 508 criteria
  - All 52 EU criteria (includes WCAG 2.1 A & AA)
  - Additional 11 WCAG 2.1 Level AAA criteria
  - Software/hardware requirements from EN 301 549

  This is a superset that satisfies US, EU, and international requirements.

  // Prisma Schema (or SQL equivalent)

  // File: prisma/schema.prisma

  model AcrEdition {
    id          String   @id @default(uuid())
    code        String   @unique // 'section508', 'wcag', 'eu', 'international'
    name        String   // 'Section 508 Edition'
    description String?
    totalCount  Int      // Total number of criteria
    createdAt   DateTime @default(now())
    updatedAt   DateTime @updatedAt

    criteria    AcrCriterion[]
  }

  model AcrCriterion {
    id          String   @id @default(uuid())
    number      String   // '1.1.1', '9.2.1', etc.
    name        String   // 'Non-text Content'
    level       String   // 'A', 'AA', 'AAA'
    description String?  @db.Text
    wcagUrl     String?  // Link to WCAG understanding document
    section     String?  // 'Perceivable', 'Operable', etc.

    // Many-to-many: A criterion can belong to multiple editions
    editions    AcrEdition[]

    createdAt   DateTime @default(now())
    updatedAt   DateTime @updatedAt

    @@unique([number, level])
  }
